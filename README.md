## Installation
### Prerequisites
* MacOS Big Sur 11.0.1 or newer
* [Command Line Tools for Xcode](https://stackoverflow.com/a/9329325)
* [Homebrew](https://brew.sh)
* [Oh My Zsh](https://ohmyz.sh)

### Basic installation

1. `git clone https://github.com/ashah01/dotfiles.git`
2. `cd dotfiles`
3. `./install.sh`
